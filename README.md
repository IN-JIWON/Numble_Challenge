# Numble Challenge_데이터로 프라이싱 전략 기반 세우기

### 📂 프로젝트 개요
기업 '화난 사람들'의 데이터를 기반해 이익 상승을 위한 프라이싱 전략을 수립하는 프로젝트입니다.

<br/>

### 📂 파일 설명
- Scrapping_Code: 화난 사람들 홈페이지 스크래핑 코드
- Final_Report: 화난 사람들 프라이싱 최종 보고서
- Excel_file: 화난 사람들 프라이싱 전략을 담은 엑셀 파일
